# s-ui backup cronjob

## Install

```sh
bash <(curl -Ls https://raw.githubusercontent.com/NimaTarlani/s-ui-backup/master/install.sh)
```
## Uninstall

```sh
bash <(curl -Ls https://raw.githubusercontent.com/NimaTarlani/s-ui-backup/master/install.sh) -u
```
